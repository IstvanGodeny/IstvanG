    Description

    This is a lightweight desktop tool built with PySide6 and SQLite to organize personal notes in a simple, intuitive interface. Users can quickly create, edit, and delete notes, each displayed as clean, card-style widgets.

    The application features a scrollable view for managing multiple notes, a confirmation system to prevent accidental loss, and a modern design using custom fonts and styles. Data is stored securely in a local SQLite database, ensuring persistence between sessions.

    This project demonstrates my skills in GUI development, database integration, and modular application design in Python.
