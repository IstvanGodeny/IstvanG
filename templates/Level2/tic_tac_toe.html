    Description

    Tic Tac Toe Game is an interactive desktop application developed with KivyMD, combining a modern material design interface with classic gameplay.

    Players can enter their names at the start, and the app dynamically tracks and displays their scores.

    The game includes smart dialogs for handling player setup, reset confirmation, quit warnings, as well as winner and draw announcements, ensuring a smooth user experience.

    A responsive design adapts the layout and font sizes to different window sizes, making the interface accessible across devices.

    The board uses color-coded symbols (X and O) to distinguish between players, with hover effects and styled buttons enhancing interactivity.

    After each round, the board resets automatically, updating scores while preserving the flow of play. Users can also fully reset the game, which clears scores and reinitializes player setup.

    This project demonstrates my ability to integrate custom UI components, responsive layouts, and event-driven logic in Python. It reflects a balance between functional gameplay and thoughtful user experience design.
