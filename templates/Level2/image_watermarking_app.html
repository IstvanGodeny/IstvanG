    _> Image Watermarking Application

    Description

    Image Watermarking Application is a desktop tool built with Tkinter and ttkbootstrap, designed to add customizable watermarks to images with ease. Users can upload images in popular formats (JPEG, PNG, GIF), preview them in real time, and apply text-based watermarks. The app supports full customization, including font family, font size, text color, and precise position adjustment using sliders. It also provides a live preview so users can fine-tune the watermark before saving. A color chooser dialog lets users pick any text color, while a font setup feature dynamically loads system or custom fonts from a dedicated folder. The application includes menu options for file management, fullscreen mode, and theme selection, giving it a polished, professional feel. With a responsive layout and user-friendly controls, it balances functionality and design for both casual and professional use. Once finished, images can be saved with the watermark applied at the exact chosen position and scale. This project demonstrates my skills in GUI development, image manipulation (PIL), font management, and real-time event-driven programming.
