    _> Portfolio Website

    Description

    Portfolio Website is a Flask-based web application designed to showcase my development projects in a modern, interactive format. The site features dedicated sections for applications such as the Note Application, Tic Tac Toe Game, and Image Watermarking tool, each presented with screenshots, icons, and short descriptions. Built with Flask and Jinja2 templating, the site dynamically organizes project content while maintaining a clean and responsive design. Frontend styling relies on Bootstrap, custom CSS, and SVG icons, ensuring a professional look across devices. Interactive elements such as Lottie animations and smooth navigation improve the overall user experience. The structure follows best practices for Flask apps, separating templates, static assets, and application logic. Deployment files (Procfile and requirements) are included to support hosting on cloud platforms like Heroku. This project demonstrates my ability to combine Python backend development with modern frontend technologies to deliver a polished portfolio site. By integrating visuals, project metadata, and structured navigation, the portfolio communicates technical skills and creative presentation effectively.
